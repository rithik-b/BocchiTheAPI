Bocchle is a daily game where you have to guess the episode of **Bocchi the Rock** on screen.

You get **6 tries** to guess the right episode using the keypad. Type your answer in 2 digits (**06** for episode 6 for example).

If the screenshot is from an ED, you can type any episode that ED appears in (like **01, 02 or 03** for ED1).

Please enjoy!
